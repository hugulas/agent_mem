# get-hermes.ai/memory/ provider selection guide

> Selection guide | 2026

## 1. Core Claim
- Hermes' 8 memory providers each excel at different use cases; this guide provides a decision matrix based on benchmarks, pricing, hosting models, and license risks, with explicit caveats about benchmark comparability.

## 2. Method / Mechanism
- **Comparison matrix**: 8 providers (Mem0, Hindsight, ByteRover, Supermemory, Holographic, OpenViking, Honcho, RetainDB) compared across:
  - Best-for use case
  - Free tier limits
  - Paid pricing
  - Hosting model (cloud / self-host / local)
  - License (Apache 2.0, MIT, AGPL-3.0, proprietary)
- **Provider profiles**: One-paragraph architecture summaries, config keys, tool counts.

## 3. Key Evidence
| Provider | Key Benchmark | Price |
|----------|---------------|-------|
| Mem0 | LongMemEval-S 67.6% | Free / $19 / $249 |
| Hindsight | LongMemEval 91.4–94.6% | Local free / Cloud pay-per-use |
| ByteRover | LoCoMo 92.2% | Local free / $19 Pro |
| Supermemory | Search-heavy focus | $19 Pro / $399 Scale |
| Holographic | Fully local, free | Free |
| OpenViking | Self-host, free | Free |
| Honcho | User modeling | $100 free credits |
| RetainDB | Structured schema recall | $20/mo Pro |

- Explicit note: "Benchmarks are not directly comparable. LoCoMo, LongMemEval, LongMemEval-S, and BEAM test different tasks on different data. Use as directional guidance only."

## 4. Limitations & Gaps
- Some benchmark scores come from vendor homepages (e.g., Hindsight 94.6% from Vectorize), not independent evaluations.
- ByteRover's LoCoMo 92.2% differs from S082's 96.1% — possible different model configs or test subsets.
- RetainDB is listed as "Cloud only / Proprietary," contradicting S087's open-source/self-hostable status.
- Does not evaluate multi-provider composition (only one provider can be active in Hermes at a time).
