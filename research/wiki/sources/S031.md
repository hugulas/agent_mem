# Skywork: Deep Dive CVE-2026-25253

> Security analysis | 2026-02-01

## 1. Core Claim
CVE-2026-25253 is a critical vulnerability (CVSS 9.8) in OpenClaw's WebSocket gateway that enables gatewayUrl manipulation, leading to token exfiltration and one-click remote code execution (RCE).

## 2. Method / Mechanism
- Attack vector: WebSocket gatewayUrl parameter manipulation
- Impact chain: crafted gateway URL → token exfiltration → authenticated access → remote code execution
- Severity: CVSS 9.8 (critical)
- Attack complexity: "one-click" implies minimal user interaction required

## 3. Key Evidence
- CVE identifier: CVE-2026-25253
- CVSS score: 9.8
- Attack primitives: gatewayUrl manipulation, token exfiltration, one-click RCE
- Target: OpenClaw WebSocket gateway component

## 4. Limitations & Gaps
- No technical details on the root cause (e.g., missing URL validation, SSRF, open redirect?)
- No patch status or affected version range disclosed
- "One-click RCE" description lacks reproduction steps or proof-of-concept details
- Source is a security blog, not a formal CVE advisory or vendor disclosure
- No discussion of whether the flaw affects memory data specifically or just gateway auth

## 5. Relevance to Our Review
Critical source for the "Security risks" section, illustrating that OpenClaw's local-first architecture still has severe remote attack surface via its WebSocket gateway.
