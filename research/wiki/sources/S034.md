# ClaudeMem vs Context Mode

> Plugin comparison | 2026-05-04

## 1. Core Claim
ClaudeMem is a community-developed memory plugin for Claude Code that adds explicit memory management commands (/remember, /forget, /memories) and a persistent memory file, supplementing Claude Code's built-in auto-memory with user-controlled memory operations.

## 2. Method / Mechanism
- Plugin architecture: extends Claude Code with custom slash commands
- Explicit memory operations: /remember to save, /forget to delete, /memories to list
- Persistent storage: dedicated memory file on disk
- Complements Claude Code's native auto-memory (S039) with manual user control

## 3. Key Evidence
- Commands: /remember, /forget, /memories
- Storage: persistent memory file
- Target platform: Claude Code
- Category: community plugin (not official Anthropic product)
