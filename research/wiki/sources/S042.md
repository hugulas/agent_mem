# ByteDance Seed blog: UI-TARS-1.5 open source

> Official blog | 2025-04

## 1. Core Claim
- UI-TARS-1.5 is a native agent model that shifts agent evolution from hand-crafted frameworks to an end-to-end trained model, integrating perception, reasoning, memory, and action into a single architecture.
- It introduces System 1 (fast, intuitive) and System 2 (slow, deliberate) reasoning modes, with a "think-before-act" mechanism that significantly improves generalization in unfamiliar GUI environments.

## 2. Method / Mechanism
- **Native agent model**: Unified perception-reasoning-memory-action structure trained end-to-end, eliminating reliance on manual rules and prompt engineering.
- **System 2 reasoning**: Reinforcement learning-based "think-then-act" module enables the model to deliberate before executing actions.
- **Self-evolving training paradigm**: Continuous accumulation of knowledge and experience during training through a data-driven closed loop.
- **Four technical pillars**: Enhanced visual perception, System 2 reasoning mechanism, unified action modeling, and self-evolving training.

## 3. Key Evidence
- Achieved **61.6%** on ScreenSpotPro benchmark, surpassing Claude (27.7%), CUA (23.4%), and previous SOTA (43.6%).
- Demonstrated stable scalability across 14 mini-games from poki.com (up to 1000 interaction steps per game).
- Achieved highest success rate in MineRL standard evaluation tasks (finding/destroying blocks, defeating hostile creatures) when the thinking module was enabled.
- Open-sourced under Apache 2.0 with GitHub repository and website.

## 4. Limitations & Gaps
- Blog post is promotional; no discussion of failure modes, safety evaluations, or adversarial robustness.
- Memory module details are sparse—no clarity on how long-term episodic memory is implemented vs. simple multi-turn context.
- Real-world deployment latency and compute requirements for the 7B+ model on consumer hardware are not disclosed.
