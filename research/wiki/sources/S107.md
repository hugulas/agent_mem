# Adaptive Memory Structures for LLM Agents

> Academic paper | 2026-02-15

## 1. Core Claim
- Existing agent memory systems rely on a single, fixed memory structure, which limits their ability to handle heterogeneous interaction patterns (relational, temporal, topical) and produces suboptimal performance.
- Adaptive memory organization—equipping agents with multiple complementary structures and learning to select among them based on interaction-level features—significantly improves long-horizon performance.

## 2. Method / Mechanism
- **FLUXMEM framework**: Equips agents with multiple complementary memory structures (flat retrieval-based, explicitly structured/graph-like, policy-managed).
- **Adaptive selection**: Learns to select memory structures based on interaction-level features using offline supervision derived from downstream response quality and memory utilization.
- **Three-level memory hierarchy**: Supports robust long-horizon memory evolution.
- **Beta Mixture Model (BMM)-based probabilistic gate**: Replaces brittle similarity thresholds with distribution-aware memory fusion.

## 3. Key Evidence
- **PERSONAMEM benchmark**: Average improvement of 9.18% over fixed-structure baselines.
- **LoCoMo benchmark**: Average improvement of 6.14%.
- Demonstrates robust performance across diverse structural patterns (relational, temporal, topical interactions).

## 4. Limitations & Gaps
- Offline supervision requires labeled interaction data and response-quality annotations, which may be expensive to collect.
- The selection model may not generalize to entirely novel interaction types not represented in training.
- BMM probabilistic gate adds computational overhead compared to simple similarity thresholds.
- Evaluation is limited to two conversational benchmarks; task-oriented agents are not tested.
