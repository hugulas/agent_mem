> Deployment guide | 2026-04-23

## 1. Core Claim
Ollama's official DeepSeek deployment rules provide granular model-size-to-RAM mappings: 1.5B models need 4 GB RAM, 7B models need 8 GB VRAM/16 GB RAM, 14B needs 12 GB VRAM/32 GB RAM, 32B needs 24 GB VRAM/64 GB RAM, and 70B needs 48 GB VRAM/128 GB RAM.

## 2. Method / Mechanism
- Quantified hardware requirements by parameter count tiers
- Dual-track guidance: discrete GPU VRAM and system RAM (or unified memory)
- Assumes standard quantization levels typical for Ollama distribution
- Linear scaling trend: each doubling of parameters roughly doubles memory requirements

## 3. Key Evidence
- 1.5B: 4 GB RAM
- 7B: 8 GB VRAM / 16 GB RAM
- 14B: 12 GB VRAM / 32 GB RAM
- 32B: 24 GB VRAM / 64 GB RAM
- 70B: 48 GB VRAM / 128 GB RAM

## 4. Limitations & Gaps
- No quantization level specified (Q4_K_M assumed but not stated)
- No tokens/second performance data at each tier
- RAM figures may not include OS and application overhead
- No guidance on CPU-only vs GPU-accelerated inference trade-offs
- Missing: multi-GPU or tensor-parallel deployment requirements for 70B+
