# Codex CLI Windows install (itecsonline)

> Install guide | 2026-03-05

## 1. Core Claim
Codex CLI can be installed natively on Windows 10 build 19041+ with 2 GB minimum RAM (4 GB recommended), Node.js 22+, and includes an experimental AppContainer sandbox for process isolation.

## 2. Method / Mechanism
- Native Windows 10 support starting from build 19041 (May 2020 Update / 20H1)
- Node.js 22+ runtime requirement
- RAM: 2 GB minimum / 4 GB recommended
- AppContainer sandbox: experimental Windows security feature for process isolation
- Installation through standard Node.js package managers

## 3. Key Evidence
- Windows 10 build 19041+ required
- 2 GB minimum RAM / 4 GB recommended
- Node.js 22+ required
- AppContainer sandbox: experimental

## 4. Limitations & Gaps
- Conflicts with S023's claim that Windows requires WSL
- AppContainer sandbox is experimental, so security guarantees are unclear
- No discussion of memory system behavior on Windows vs. macOS/Linux
- No guidance on Windows-specific path handling for AGENTS.md or memory files
- Missing: whether the Rust binary (mentioned in S023) is available on Windows
