# Jiang et al. (2026): Anatomy of Agentic Memory

> Academic paper | 2026-02

## 1. Core Claim
- Agentic memory systems consistently underperform their theoretical promise because empirical foundations are fragile: benchmarks are underscaled relative to modern context windows, evaluation metrics misalign with semantic utility, accuracy varies significantly across backbone models, and system-level costs (latency, throughput) are frequently overlooked.
- A principled taxonomy of four memory structures (Lightweight Semantic, Entity-Centric and Personalized, Episodic and Reflective, Structured and Hierarchical) can expose why complex memory systems are often tested in settings where simpler full-context or retrieval baselines suffice.

## 2. Method / Mechanism
- **Taxonomy-driven survey**: Organizes Memory-Augmented Generation (MAG) systems around four memory structures defined by how memory is organized and manipulated.
- **Empirical bottleneck analysis**: Connects each memory structure to specific pain points—benchmark saturation effects, metric validity, LLM-as-a-judge prompt sensitivity, backbone-dependent accuracy, and latency/throughput overhead from memory maintenance.
- **System perspective**: Analyzes not just algorithmic accuracy but end-to-end system costs, including the overhead of maintaining memory stores across interactions.

## 3. Key Evidence
- Identifies that reported gains across agentic memory systems are inconsistent across benchmarks and highly backbone-dependent.
- Documents benchmark saturation: complex memory systems are tested on benchmarks where simpler full-context or retrieval baselines already suffice, obscuring true benefits.
- Highlights that metrics often emphasize surface overlap (e.g., F1) rather than semantic utility, and LLM-as-a-judge evaluations are sensitive to prompt wording.
- System-level costs (latency, throughput degradation from memory maintenance) are rarely measured in existing literature.

## 4. Limitations & Gaps
- As a survey, the paper diagnoses problems rather than proposing or validating a new architecture.
- The critique of backbone-dependent accuracy may itself vary with newer model releases.
- Does not provide a unified benchmark or evaluation protocol to replace the fragmented existing ones.
- System cost analysis is observational; no controlled experiments isolate the cost of specific memory operations.

## 5. Relevance to Our Review
- **Academic memory architectures**: Provides the foundational taxonomy for structuring our review of agent memory systems.
- **Benchmarks / Long-term memory evaluation**: Critical source for the evaluation-gap subsection; documents why current benchmarks may not meaningfully distinguish memory architectures.
