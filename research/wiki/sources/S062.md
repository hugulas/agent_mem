## 1. Core Claim
- AI-generated personalized music (Context-AI Tune) that adapts to both the user's environment and self-assessed stress level is more effective at reducing stress than manually chosen pre-recorded relaxing music.

## 2. Method / Mechanism
- **System**: Context-AI Tune (CAT) — a web application with Vue 3 frontend and Python Flask backend.
- **Inputs**: Camera captures environment → ChatGPT VLM extracts descriptive keywords (e.g., "trees," "building") → user selects keywords and sets stress level via slider → Suno API generates adaptive music tracks.
- **Architecture**: Dual-component (Vue 3 frontend + Flask backend); images encoded as Base64; SQLite database logs interactions; CSV export for analysis.
- **Experiment**: 2×2 within-subject design (AI vs. NoAI × Busy Hub vs. Quiet Library) with N=26 participants.

## 3. Key Evidence
- N=26 within-subject experiment using Visual Analog Scale for Stress (VAS-S).
- Stress-inducing math tasks with timed problem solving, auditory feedback, and leaderboards.
- Results: CAT outperformed manually chosen music in stress reduction across both environments.
- Systematic workflow figures and UI mockups documented.

## 4. Limitations & Gaps
- Small sample size (N=26); limited generalizability.
- Stress measurement relies on self-reported VAS-S rather than physiological biomarkers.
- Music generation depends on third-party Suno API — reproducibility and availability risks.
- Only two environment types tested (busy hub vs. quiet library).
