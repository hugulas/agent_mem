# Claude Code hardware (bswen)

> Community guide | 2026-03-11

## 1. Core Claim
Claude Code has **minimal hardware requirements** because it offloads all LLM inference to cloud APIs: any CPU, 2 GB RAM, and Node.js 18+ are sufficient, with Raspberry Pi confirmed working and no GPU needed.

## 2. Method / Mechanism
- **Minimum specs**: Any CPU / 2 GB RAM / Node.js 18+.
- **No GPU required**: All inference happens via cloud APIs (Anthropic).
- **Raspberry Pi confirmed working**: Demonstrates extreme minimalism.
- **Lightweight local runtime**: The CLI is essentially a thin harness around API calls.

## 3. Key Evidence
- Any CPU sufficient.
- 2 GB RAM minimum.
- Node.js 18+ required.
- Raspberry Pi explicitly confirmed as working.
- No GPU needed.

## 4. Limitations & Gaps
- **Source is an nginx 404 error page**. Content from reading-log key claims only.
- No information on network bandwidth requirements (critical for streaming API responses).
- No discussion of disk space for session transcripts or CLAUDE.md files.
- Raspberry Pi "confirmed working" but no model specified (Pi 3? 4? 5? Performance will vary).
- Local LLM setup (Ollama + Claude Code) not covered — would dramatically change requirements.

## 5. Relevance to Our Review
- **Hardware requirements** — Claude Code minimal specs
- **Cloud-native design** — contrast with local-inference systems (OpenClaw, Hermes)
- **Accessibility** — low barrier to entry for individual developers
