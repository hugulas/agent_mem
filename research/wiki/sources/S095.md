# Gamgee: Mem0 vs Supermemory comparison

> Comparison blog | 2026-04-16

## 1. Core Claim
- Mem0 and Supermemory target different maturity stages: Mem0 offers the largest community, broadest framework ecosystem, and open-source self-hosting; Supermemory offers stronger benchmark performance, built-in connectors, and multi-modal support.

## 2. Method / Mechanism
- **Mem0**: Vector + knowledge graph hybrid; LLM-based fact extraction; semantic retrieval; optional graph memory via `pip install mem0ai[graph]`; Apache 2.0.
- **Supermemory**: Hybrid RAG with structured fact extraction; handles temporal changes and contradictions; connectors for Google Drive, Notion, Gmail, GitHub; multi-modal (PDF, image OCR, video transcription, code).

## 3. Key Evidence
| Factor | Mem0 | Supermemory |
|--------|------|-------------|
| LongMemEval | 49% | 85.2% |
| Deployment | Cloud or self-hosted | Cloud API only |
| Pricing | Free / $19 / $249 / Enterprise | Not disclosed |
| GitHub stars | 52.8K | 21.7K |
| Funding | $24M Series A | $2.6M seed |
| License | Apache 2.0 | Proprietary |

- Mem0 has first-class integrations with CrewAI, Flowise, Langflow, AWS Strands.
- Supermemory retrieval latency ~50ms.

## 4. Limitations & Gaps
- Mem0's 49% LongMemEval figure is likely outdated; S078 shows Mem0's 2026 algorithm achieves 94.4% on LongMemEval and 92.5% on LoCoMo.
- Comparison may use different judge configurations or model backends, making direct score comparison unreliable.
- Supermemory's undisclosed pricing is a genuine barrier for evaluators.
- No discussion of privacy, data residency, or compliance.
