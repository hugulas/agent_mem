# SkillSieve: A Hierarchical Triage Framework for Detecting Malicious AI Agent Skills

> Academic paper | 2026-04-08 | arXiv:2604.06550

## 1. Core Claim
OpenClaw's ClawHub is the largest public skill registry (13,000+ skills as of Feb 2026). Security audits reveal that **13–26% of skills contain security vulnerabilities**, and the ClawHavoc campaign demonstrated coordinated supply-chain attacks. Existing scanners (regex-based ClawVet, formal-static SkillFortify, single-model VirusTotal) each cover only part of the problem because skills are dual-modal: code + prose. SkillSieve proposes a three-layer triage framework that achieves **0.800 F1** on a 400-skill labeled benchmark at **$0.006 per skill**.

## 2. Method / Mechanism
- **Layer 1 (Static triage)**: Regex + AST + XGBoost feature scorer filters ~86% of benign skills in <40ms at zero API cost
- **Layer 2 (Decomposed LLM analysis)**: Four parallel sub-tasks (intent alignment, permission justification, covert behavior detection, cross-file consistency) with structured output
- **Layer 3 (Jury verdict)**: Three different LLMs vote independently; if they disagree, they debate before reaching consensus
- **Dataset**: 49,592 real ClawHub skills + 400 labeled benchmark + five evasion techniques
- **Hardware**: Full pipeline runs on a $440 ARM single-board computer in 31 minutes

## 3. Key Evidence
- **ClawHub vulnerability rates** (cited from prior audits):
  - Snyk ToxicSkills: **13.4%** of 3,984 skills contain at least one critical-level issue; **36.82%** have at least one flaw of any severity
  - Liu et al. (agent-skills-in-the-wild): **26.1%** of 42,447 skills contain vulnerabilities
  - Koi Security audit of 2,857 skills: 341 malicious entries (11.9%), 335 tied to ClawHavoc campaign
  - ClawHavoc campaign: hundreds of malicious skills pushed over six weeks
- **SkillSieve performance**: 0.800 F1 (0.752 precision, 0.854 recall) on 400-skill benchmark; outperforms ClawVet (0.421 F1)
- **Layer 1 alone**: 0.989 recall at zero cost
- **Cost**: $0.006 per skill average

## 4. Limitations & Gaps
- Layer 1 cannot catch runtime-fetched payloads (remote URL at execution time)
- LLM outputs are non-deterministic (temperature=0, 3-run averaging)
- Framework currently targets OpenClaw skills; transfer to MCP servers/LangChain tools requires new rule sets
- Edge deployment on $440 ARM board is a proof-of-concept, not a production deployment
