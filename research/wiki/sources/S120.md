# TiMem: Temporal-Hierarchical Memory Consolidation

> Academic paper | 2026-01

## 1. Core Claim
- Existing memory frameworks provide limited support for temporally structured information across hierarchical levels, leading to fragmented memories and unstable long-horizon personalization.
- TiMem treats temporal continuity as a first-class organizing principle, using a Temporal Memory Tree (TMT) with semantic-guided consolidation and complexity-aware recall to achieve state-of-the-art accuracy while reducing recalled memory length by over 50%.

## 2. Method / Mechanism
- **Temporal Memory Tree (TMT)**: Five-level hierarchy organizing conversations from raw observations to abstracted persona profiles (Segments → Session → Daily → Weekly → Profile).
- **Semantic-guided consolidation**: Enables memory integration across hierarchical levels without fine-tuning, using LLM-based semantic merging.
- **Complexity-aware memory recall**: Balances precision and efficiency by adapting recall depth to query complexity (Simple / Hybrid / Complex modes).
- **Temporal sliding window**: Organizes dialog streams along a time axis with automatic consolidation triggers.

## 3. Key Evidence
- **LoCoMo**: **75.30%** accuracy (SOTA among evaluated baselines).
- **LongMemEval-S**: **76.88%** accuracy (SOTA).
- Reduces recalled memory length by **52.20%** on LoCoMo.
- Manifold analysis indicates clear persona separation on LoCoMo and reduced dispersion on LongMemEval-S.
- Outperforms baselines including GPT-4o-mini, GPT-4o, and Qwen3-8B under consistent evaluation setup.

## 4. Limitations & Gaps
- Five-level hierarchy may be over-engineered for short-horizon or single-session tasks.
- Consolidation quality depends on the LLM used for semantic merging; weaker models may produce lower-quality abstractions.
- The TMT structure assumes calendar-time organization (daily/weekly); agents with non-calendar interaction patterns may not fit neatly.
- Evaluation is on conversational benchmarks; generalization to task-oriented or coding agents is not demonstrated.

## 5. Relevance to Our Review
- **Hierarchical memory**: The most explicit temporal-hierarchical architecture in our surveyed sources.
- **Long-term memory evaluation**: Strong SOTA results on LoCoMo and LongMemEval-S provide credible empirical support.
- **Academic memory architectures**: Temporal continuity as a first-class principle is a useful conceptual contribution.
