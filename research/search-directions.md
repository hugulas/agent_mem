# Search Directions

## Research Frame

- **Exact research question:** How do mainstream agent frameworks (OpenClaw, Claude Code, Codex CLI, Hermes Agent) implement memory systems? What technologies, architectures, and data stores do they use? What are their system performance requirements and chip/hardware demands?
- **Scope boundaries:** Focus on the memory subsystem of each framework—persistent storage, context management, retrieval mechanisms, compaction/compression, and cross-session continuity. Exclude general LLM training, model architecture, or non-agent applications unless directly relevant to memory.
- **Time boundary:** Prefer sources from 2025–2026. Older sources accepted only for foundational concepts.
- **Languages allowed:** English and Chinese.
- **Source preferences:** Official documentation, engineering blogs, arXiv papers, reverse-engineering reports, GitHub repositories, benchmark studies.
- **Exclusion rules:** Exclude pure marketing without technical substance; exclude sources that merely repeat official docs without added analysis.
