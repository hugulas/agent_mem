## 1. Core Claim
- ByteRover CLI 0.3.1 (beta) introduces Context Tree and Agentic Search, replacing flat vector-DB organization with a hierarchical, agent-driven retrieval system that dramatically improves memory retrieval quality.

## 2. Method / Mechanism
- **Context Tree**: Hierarchical memory structure — Domains (high-level categories) → Topics (specific subjects) → Context Files (Markdown with actual knowledge).
- **Agentic Search**: Replaces vector DB + similarity search; the agent navigates the codebase like a developer would, using tool use and structured data to retrieve exact context pieces rather than full chunks.
- **Team sync**: New `brv pull` command to sync latest memory updates from teammates.
- **Cleaner core commands**: Redesigned CLI UX.
