## 1. Core Claim
- Mem0 and Supermemory target different maturity stages: Mem0 offers the largest community, broadest framework ecosystem, and open-source self-hosting; Supermemory offers stronger benchmark performance, built-in connectors, and multi-modal support.

## 2. Method / Mechanism
- **Mem0**: Vector + knowledge graph hybrid; LLM-based fact extraction; semantic retrieval; optional graph memory via `pip install mem0ai[graph]`; Apache 2.0.
- **Supermemory**: Hybrid RAG with structured fact extraction; handles temporal changes and contradictions; connectors for Google Drive, Notion, Gmail, GitHub; multi-modal (PDF, image OCR, video transcription, code).
