## 1. Core Claim
- Honcho enhances Hermes's built-in memory with dialectic reasoning and deep user modeling, using a two-layer context injection system with cold/warm prompt selection and three orthogonal configuration knobs for cost/depth control.

## 2. Method / Mechanism
- **Two-Layer Context Injection**:
  1. **Base context** — session summary, user representation, user peer card, AI self-representation, AI identity card. Refreshed on `contextCadence`.
  2. **Dialectic supplement** — LLM-synthesized reasoning about user's current state and needs. Refreshed on `dialecticCadence`.
- **Cold/Warm Prompt Selection**:
  - **Cold start** (no base context yet): General query — "Who is this person? What are their preferences, goals, and working style?"
  - **Warm session** (base context exists): Session-scoped query — "Given what's been discussed in this session so far, what context about this user is most relevant?"
- **Three Orthogonal Config Knobs**:
  - `contextCadence` (default 1): Turns between base-layer refresh.
  - `dialecticCadence` (default 2, recommended 1–5): Turns between dialectic LLM calls.
  - `dialecticDepth` (default 1, clamped 1–3): Number of `.chat()` passes per dialectic invocation.
- **Dialectic Depth (Multi-Pass)**:
  - Pass 0: Cold or warm prompt.
  - Pass 1: Self-audit — identifies gaps and synthesizes evidence from recent sessions.
  - Pass 2: Reconciliation — checks for contradictions and produces final synthesis.
- **Query-Adaptive Reasoning**: Auto-scales `dialecticReasoningLevel` by query length (+1 at ≥120 chars, +2 at ≥400).
- **Session-Start Prewarm**: Background dialectic call at full depth on session init.

## 4. Limitations & Gaps

- Cold/warm selection is automatic but could misfire if base context is stale or incomplete.

## 7. Personal Annotation
- The orthogonality of the three knobs is a well-designed UX — users can tune frequency and depth independently.
- The cold/warm distinction is a simple but effective optimization; it avoids expensive general queries once a user model exists.
