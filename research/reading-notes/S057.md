## 1. Core Claim
- Apple Intelligence is a personal intelligence system built into iPhone, iPad, Mac, and Vision Pro that prioritizes on-device processing (3B Apple Foundation Models), Private Cloud Compute for complex queries, and a semantic index for personal context—designed with privacy at every step.

## 2. Method / Mechanism
- **On-device processing**: Apple Foundation Models (~3B parameters) run locally on A17 Pro/M-series chips with Neural Engine and Apple Silicon optimizations.
- **Private Cloud Compute**: Extends processing to Apple-owned servers for complex queries without storing or exposing personal data.
- **Semantic index**: Organizes personal context (emails, messages, photos, calendar) for retrieval without cloud aggregation.
- **App Intents**: Standardized toolbox for apps to expose actions to the system intelligence layer.
- **Orchestration layer**: Routes queries between on-device model, Private Cloud Compute, and ChatGPT integration (with explicit user permission).
- **ChatGPT integration**: Optional opt-in to GPT-4o for queries beyond Apple Intelligence's capabilities.

## 3. Key Evidence
- **3B on-device model**: Runs on iPhone 15 Pro and later, iPad/Mac with M-series chips.
- **Live Translation**, **Writing Tools**, **Image Playground**, **Genmoji**, **priority notifications** as showcased features.
- **Visual Intelligence**: Camera-based contextual awareness for identifying objects and places.
- **Siri enhancements**: Improved natural language understanding and cross-app action capabilities.

## 4. Limitations & Gaps
- Marketing page, not technical documentation—no details on model architecture, quantization, or benchmark scores.
- Hardware requirements (A17 Pro/M-series) exclude older devices, limiting accessibility.
- Semantic index details are sparse—unclear if this is a vector DB, knowledge graph, or keyword index.
- Siri overhaul and advanced personal context features have been delayed to 2026 (per S059).
- ChatGPT integration is a dependency on a third-party service, complicating the "privacy-first" narrative.
