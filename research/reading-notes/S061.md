## 1. Core Claim
- A cross-vendor comparison of memory capabilities across major AI providers (2024–2025) reveals divergent strategies: OpenAI offers editable session memory, Anthropic provides persistent memory in Claude 3.5/4, Google focuses on long-term document memory, Microsoft emphasizes task state memory, and Meta is experimenting with persistent agents.

## 2. Method / Mechanism
- **OpenAI**: Session memory that users can edit; integrated into ChatGPT interactions.
- **Anthropic**: Persistent memory in Claude 3.5 and Claude 4; stores facts and preferences across conversations.
- **Google**: Long-term document memory via large context windows and context caching in Gemini.
- **Microsoft**: Task state memory integrated into Copilot workflows; emphasizes enterprise governance.
- **Meta**: Experimental persistent agents using Llama models with framework integrations.
